# Plentific

Plentific Challenge

### Installing

```sh
$ git clone https://gitlab.com/seechrobin/plentific-coding-challenge
$ cd plentific-coding-challenge
$ yarn install
```

## Running

```sh
$ yarn start
```

## Testing

```sh
$ yarn test
```
